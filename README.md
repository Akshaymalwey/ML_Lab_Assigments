# ML Lab Assignments

This repository contains several machine learning notebooks and related lab exercises. Each notebook is summarized below with the core topic, datasets, and techniques used.

- `ML_Lab1.ipynb`: Introductory notebook focused on NumPy fundamentals and basic array operations. It likely builds essential numerical programming skills used in later ML labs. This is the foundational first lab for data preparation and matrix handling.


- `ML_Lab2 (1).ipynb`: Supervised learning lab using logistic regression with model training, prediction, and evaluation. The notebook includes calculation of accuracy and confusion matrix metrics. It is a practical binary classification exercise with real dataset handling.


- `ML_lab3.ipynb`: Polynomial regression lab that trains `LinearRegression` on transformed polynomial features. The notebook reads training data, fits models, and visualizes results. It demonstrates overfitting/underfitting behavior and regression curve fitting.


- `ML_Lab4 (1).ipynb`: Regression analysis with data loading, exploratory plots, and model predictions. It appears to load an ice cream sales dataset and use polynomial or linear regression for trend modeling. The lab also includes visualization of confidence interval trends.


- `MLLab4_b.ipynb`: Linear algebra utilities and matrix operations implementation. This notebook defines functions for transpose, multiplication, and determinants, reinforcing matrix computation fundamentals. It is a programming-focused mathematical exercise rather than a full ML pipeline.


- `ml_lab_bias_variance_tradeoff.ipynb`: Bias-variance lab using a salary dataset and regression modeling. The notebook compares simple and polynomial fits to illustrate generalization and model complexity. It likely includes plots showing how error changes with model degree.


- `Lab_5_Logistic_Regression.ipynb`: Logistic regression lab that generates synthetic classification data with `make_classification` and `make_moons`. It fits models, computes accuracy, and visualizes decision boundaries. The focus is on classification performance and dataset separability.


- `LogisticRegression_LinearAndNonLinearDataset (2).ipynb`: Demonstrates logistic regression on both linear and non-linear datasets. It includes data generation, model training, and comparison of classifier behavior across dataset shapes. The notebook is useful for understanding how logistic regression handles different decision surfaces.


- `neural_network.ipynb`: Neural network notebook that loads training data from CSV and builds a deep learning model. It includes training over multiple epochs and prediction steps. The file shows practical work with network training and feature preprocessing.


- `SVM (1).ipynb`: Support Vector Machine lab using both custom optimization and scikit-learn implementation. It generates blob datasets, fits SVM models, and evaluates classification accuracy. The notebook covers linear separability, margins, and support vectors.
